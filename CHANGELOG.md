# Changelog

## 2.0.0

### 主要更新
- 重做了字幕悬浮窗，让块状字幕更稳定，堆叠显示更自然。
- 优化了字幕块插入和流式更新过程，减少抖动和跳跃。
- 改进了保存流程，修复保存按钮无反应的问题。
- 在悬浮窗中加入了 token 统计，并补充了更清晰的启动进度提示。
- 调整了翻译请求策略，让首译更快、长句不容易卡住。
- 增加了暂停/继续功能，并优化了跟随最新字幕的滚动逻辑。
- 增加了自动检查 GitHub Release 更新的功能，支持配置检查仓库。

### 打包信息
- 发布版本号更新为 `2.0.0`。
- DMG 和应用包命名已同步为新的版本线。

### 说明
- API Key 和本地配置不会被打进发布包。
- 打包后的程序会优先读取用户目录下的本地配置。
